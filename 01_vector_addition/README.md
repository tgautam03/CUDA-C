# Vector Addition

## Problem Statement
### Inputs
- `N`: Vector length
- `A`: Vector A of length N
- `B`: Vector B of length N
- `C`: Vector C of length N

### Operation
`C = A + B`

### Compilation and execution
To compile run `make vec_add` in the terminal. This is create an executable `vec_add.out`.
